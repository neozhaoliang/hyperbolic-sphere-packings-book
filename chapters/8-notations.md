# 记号

|   |      |
|:---:|:---:|
|  $W$  |  Coxeter 群   |
|  $\Gamma$ | Coxeter 图 |
|  $\cl{X}$ | 集合 $X$ 的闭包 |
|  $X^\circ$ | 集合 $X$ 的内点 |
|  $\cone{X}$ | $\{\sum c_ix_i\mid x_i\in X,\ c_i\geq0\}$   |
| $\lfun{v}{f}$ | 双线性对 $V\times V^\ast\to\R: \lfun{v}{f}\to f(v)$ |
|  $\Delta=\{\alpha_s\}$       |   单根系       |
|  $\Delta^\ast=\{\omega_s\}$   |   $\Delta$ 的对偶基，即基本权 |
|  $\Q$       |    双曲空间中 $(v,v)\leq0$ 的点组成的集合  |
|  $u\sim v$    |   $u,v$ 属于 $\minus{\Q}{0}$ 的同一个连通分支 |
